# Sistema de Suporte de TI (Help Desk)

## Descrição
Projeto desenvolvido para simular um sistema básico de suporte técnico (Help Desk), permitindo o registro e acompanhamento de chamados.

## Funcionalidades
- Abertura de chamados
- Alteração de status (Aberto, Em andamento, Resolvido)
- Interface simples e funcional

## Tecnologias
- HTML
- CSS
- JavaScript

## Objetivo
Simular um ambiente real de suporte técnico, aplicando conceitos de resolução de problemas e organização de chamados.
